# APSIM Registration System

## Overview

This is the APSIM Registration System repository. For detailed information about specific projects, please refer to the project-level documentation below.

## Project Documentation

- **[RegistrationShared](./RegistrationShared/README.md)** - Shared class library for Web App and Web API
- **[RegistrationWebApi](./RegistrationWebAPI/README.md)** - C# Mimimal API for handling APSIM registrations
- **[RegistrationWebApp](./RegistrationWebApp/README.md)** - C# Blazor Web Assembly Web Application that enables user registrations and downloads of APSIM Next Gen and APSIM Classic releases.
